# emacs-kinopoisk

## Summary

API of `kinopoisk` (cinema-service) for Emacs Lisp

## Installing

You will need Emacs 24+, `make` and [Cask](https://github.com/cask/cask) to
build the project.

    cd emacs-kinopoisk
    make && make install
